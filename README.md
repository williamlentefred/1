# WeDeliverTECH Reception Management Dashboard

## Beskrivelse
En web applikasjon for WeDeliverTECH som lar resepsjonisten håndtere ansatte og leveringer.

## Filstruktur
- Web Application/index.html - Hovedsiden
- Web Application/style.css - Styling
- Web Application/wdt_app.js - JavaScript
- Documentation/Jira.pdf - Prosjektrapport

## Slik kjører du applikasjonen
1. Åpne mappen i VS Code
2. Høyreklikk på index.html
3. Velg "Open with Live Server"
4. Applikasjonen åpnes i nettleseren

## Eksterne biblioteker
- Bootstrap 5.3.0 - https://getbootstrap.com
- jQuery 3.6.0 - https://jquery.com
